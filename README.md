# Optimizaci-n-Rampa-Aeroportuaria-
Estoy desarrollando un sistema de optimización de rotación de personal de rampa inspirado en un problema real que viví trabajando en Aeroparque. Observé asignaciones manuales, demoras y descansos desiguales. El proyecto modela grupos operativos y asigna tareas de forma automática y equitativa, mejorando la organización y la trazabilidad del trabajo
